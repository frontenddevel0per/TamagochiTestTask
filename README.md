# Tamagochi Test Task

### Used stack:

- TypeScript
- Next.js
- React
- React Hooks
- TailWind

### How to launch:

1. Download this repository.
2. In console, open this folder.
3. Run `npm i` to install all packages.
4. Run `npm run start` to start app.
